1，Java核心技术·卷 I ~~不错入门选择

> 该书籍基本上介绍java的基本特性，继承，封装，多态的特性

![java核心技术卷1](https://blog-img-qrx.oss-cn-beijing.aliyuncs.com/img/java核心技术卷1.jpg)

如果你觉得书籍内容太过繁杂，你可以阅读JavaGuide手册，它非常突出一些重点
